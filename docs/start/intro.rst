Starting Up
-----------

This tutorial guides you through the installation process of the *Imperfect 
Collection* loader for Pump It Up arcade games. The loader provides a 
streamlined way to boot and manage game images directly on MK hardware, making 
it ideal for enthusiasts and operators seeking a quick deployment method.

The steps presented here are straightforward and can be followed by users on 
Windows, macOS, or Linux systems. Basic familiarity with creating bootable disks
and handling hardware boot procedures will help, but the tutorial is designed to
be clear and accessible regardless of your operating system.

Please read each section carefully before proceeding to ensure a smooth 
installation experience.
